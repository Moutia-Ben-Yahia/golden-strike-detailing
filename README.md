# CleanStrike Unit - Luxury Mobile Car Detailing

## About

CleanStrike Unit is a professional mobile car detailing service in Monaco, specializing in luxury and supercars. We bring showroom-quality detailing directly to you - whether you're at your residence, parking spot, or hotel.

## Development

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd golden-strike-detailing

# Install the necessary dependencies
npm install

# Start the development server
npm run dev
```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
